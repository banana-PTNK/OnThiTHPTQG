Cài đặt AI OCR (Tự động kẻ bảng)

Để hệ thống có thể tự động bóc tách và vẽ bảng chính xác 100%, bạn cần cung cấp một khóa Google Gemini API Key miễn phí. Khóa này chỉ được lưu cục bộ trên trình duyệt của bạn.

📚 Hướng dẫn lấy Key miễn phí (Chỉ mất 1 phút)

  1. Truy cập Google AI Studio.
  2. Đăng nhập bằng tài khoản Google (Gmail) của bạn.
  3. Nhấn nút "Create API key" (Màu xanh) -> Chọn Create API key in new project.
  4. Một mã bảo mật (bắt đầu bằng AIzaSy...) sẽ hiện ra. Hãy Copy (Sao chép) mã đó.
  5. Quay lại trang web này, Paste (Dán) mã vào ô phía trên và nhấn Lưu.
